Express.js Typescript Starter Boilerplate.

## Features

- Express.js
- Typescript
- Webpack
- Jest - supertest
- Swagger
- Prettier
- ESLint

## Usage

local development

```
npm start
```

---

production

```
npm run build
```

## Unit testing

```
npm test
```

```
src --

      |        |

      |        |

      |        |
```
